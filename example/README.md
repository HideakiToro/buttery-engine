# Buttery-Engine-Example
An example project showcasing and testing the buttery-engine game-engine project.

## How to build for web?
- To build for web, I followed [this guide](https://github.com/gfx-rs/wgpu/wiki/Running-on-the-Web-with-WebGPU-and-WebGL). As soon as all tools have been downloaded, you can simply run the build-for-web.sh script.

## Things of note
Due to upstream errors in slippery-renderer, web support is currently broken.
